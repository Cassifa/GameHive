========================================
不围棋RAG知识库
========================================

【规则特别强调】
不围棋与围棋的关键区别：
- 围棋：目标是围地获胜
- 不围棋：目标是避免无气死棋
- 策略重点：保持棋子的气（自由度）

【基本概念】

气的定义：
棋子上下左右相邻的空点数量
单子的气：最多4气（上下左右）
群子的气：所有相邻空点的总和（去重）

死棋条件：
一块棋子的气数降为0时，该块棋子死亡
死棋需立即从棋盘上移除

【最佳开局策略 - 边角优先】

绝对首选开局：边角组合(0,1)(1,0)
重要性：10/10 - 绝对关键
理由：
- 边角位置天然具有更高气数
- (0,1)(1,0)组合形成稳固L形基地
- 控制了左上角区域且不易被围杀
- 为后续向两个方向拓展奠定基础

四大黄金开局组合（按优先级）：
1. 左上角：(0,1)(1,0) - 最佳选择
2. 右上角：(0,5)(1,6) - 同等效果
3. 左下角：(5,0)(6,1) - 同等效果
4. 右下角：(5,6)(6,5) - 同等效果

必须避免的开局：
- 中心点(3,3)及周围区域：极易被围杀
- 单独角点(0,0)：气数过少，易受攻击

【边角开局拓展战略】

边角基地拓展路径：
第1-2手：占据(0,1)(1,0)建立基本L形
第3-4手：沿边缘拓展至(0,2)(2,0)
第5-6手：巩固内部联系(1,1)(1,2)或(2,1)
第7-8手：持续沿边扩张(0,3)(3,0)

对手干扰应对：
- 若对手占据你计划中的点位：绕过并继续沿边扩展
- 若对手试图切断你的连接：立即补强关键连接点
- 若对手在中心布局：继续专注于边角发展，不要被干扰

阻挠对手占领对角的策略：
- 预先占据关键转角点(如对手占据(6,6)时，抢先占据(6,5)和(5,6))
- 在己方基地稳固后，主动出击占据远角
- 使用"长臂进攻"策略跳至对角区域建立桥头堡

【边缘优先策略详解】

边缘位置优势：
- 天然减少被包围的可能性
- 更容易保持气数
- 便于向内部扩展

角位特殊性：
- 最安全的位置
- 仅需防守两个方向
- 扩展空间充足

中心危险性：
- 容易被四面包围
- 气数容易被压缩
- 逃脱困难

【对角扩展理论】

基本原理：
通过对角线方向扩展，最大化控制范围
同时保持棋子间的连接

实施步骤：
1. 建立边角L形基地(0,1)(1,0)
2. 向对角方向发展
3. 形成稳固的连接结构

【边角L形基地详解】

L形基地核心优势：
- 形成稳固的角部控制
- 两个方向的扩展可能
- 气数共享最优化
- 难以被对手切断

L形基地气数分析：
标准L形(0,1)(1,0)：
- 单独时：各3气
- 连接后：共享气点，总计5气
- 扩展至(0,2)(2,0)后：总计7气
- 完整角部占据后：总计8-9气

L形基地防御策略：
- 保持边缘连接，避免内部孤立子
- 优先防守角部关键点
- 在受到威胁时向内部延伸增加气数

【包围与反包围】

包围技巧：
1. 识别对手孤立棋子
2. 切断其与外界的连接
3. 逐步压缩其气数空间

反包围策略：
1. 及时识别包围威胁
2. 主动突围或连接
3. 建立多个出路

【气数计算详解】

单子气数：
边角位置：2-3气
边缘位置：3气
内部位置：4气

群子气数计算：
1. 统计群子周围所有空点
2. 去除重复计算的空点
3. 得出群子总气数

危险警报：
1气：极度危险，下手即死
2气：高度危险，需要立即处理
3气：中度危险，需要关注
4气及以上：相对安全

【实战技巧】

开局原则：
1. 首选边角L形布局(0,1)(1,0)
2. 其次沿边缘扩展
3. 绝对避免中心
4. 积极阻断对手占据角位

中局要点：
1. 保持连接
2. 扩大地盘
3. 寻找对手弱点
4. 阻断对手连接路线

残局技巧：
1. 精确计算气数
2. 寻找致命一击
3. 防守反击并重
4. 优先保护边角基地
